# Conway's Game of Life
---
## usage
- building and running: `make && ./gol`<br>
	- dependencies: SDL2
	- tested on debian 10

- using the simulation:
	- `space` to pause/resume simulation
	- `r` to randomize the cells
	- left click on cells to toggle alive/dead (black = alive, white = dead)
